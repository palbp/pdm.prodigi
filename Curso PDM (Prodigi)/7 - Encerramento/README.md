# 7 - Encerramento

Ao concluir o curso, os participantes deverão ter adquirido competências para:
- preparar o ambiente de trabalho e compreender os artefactos do projecto;
- compreender a anatomia de aplicações Android;
- compreender o ciclo de vida de Activities e a navegação entre ecrãs;
- construir interfaces gráficas usando Jetpack Compose;
    - compreender o ciclo de vida de composables;
    - reagir a eventos do utilizador;
    - gerir estado da UI e distnguir estado de apresentação de estado de aplicação;
    - aplicar state hoisting para separar UI e comportamento;
- desenhar aplicações Android com arquitetura MVVM (Model View ViewModel);
- compreender e aplicar eficazmente o modelo de concorrência baseado em corrotinas;
- integrar comunicação HTTP assíncrona em aplicações Android.

Com estas competências, ficam aptos a conceber e implementar aplicações Android modernas de pequena/média complexidade e a prosseguir para temas mais avançados tais como persistência local (com DataStore ou Room), persistência remota (com Firebase), arquiteturas reativas com Flows e realização de trabalho em segundo plano.
