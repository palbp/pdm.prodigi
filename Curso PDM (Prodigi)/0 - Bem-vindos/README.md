## Sinopse

Curso introdutório ao desenvolvimento de aplicações Android. Apresentam-se práticas atuais usando a linguagem Kotlin e as APIs modernas da plataforma, nomeadamente Jetpack Compose para construção de interfaces gráficas, o padrão MVVM (Model View VIewModel) e o modelo de concorrência baseado em corrotinas. Trata-se de um curso introdutório com enfoque prático; no final, os participantes estarão aptos a conceber e implementar aplicações Android alinhadas com as práticas atuais.

## Pré-requisitos

Presume-se que os participantes têm competências de programação e que estão familiarizados com a linguagem Kotlin.

## Tópicos abordados

- Ambiente e ferramentas de desenvolvimento
- Anatomia de aplicações Android
- Activity: propósito, ciclo de vida e consequencias das reconfigurações
- Navegação entre Activities (_intents_, _user tasks_ e _back-stack_)
- Desenvolvimento de interfaces gráficas com Jetpack Compose
  - Fundamentos (funções _composable_, modificadores)
  - Gestão de estado (_composables stateless_ e _statefull_, _state hoisting_)
- Arquitetura moderna de aplicaçoes Android
  - Padrão MVVM
  - Modelo de concorrência Android
- Aplicações Android enquanto clientes HTTP
- Caso de estudo prático envolvendo integração ponta a ponta com Web API
  - Injecção de dependências
  - UI enquanto máquina de estados
  - Modelos de comunicação push e pull
- Armazenamento local persistente (introdução)
- Encerramento e próximos passos
